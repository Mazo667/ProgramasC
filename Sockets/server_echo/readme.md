<h1>SERVER ECHO</H1>
Un servidor eco <b>(echo)</b> es un tipo de servidor que simplemente reenvía de vuelta cualquier dato que recibe de un cliente.  Para eso se usa <b>dup2(int file_descrip1, int file_descrip2)</b> que copia un descriptor de archivo a otro especificando que descriptor usar para la copia. Entonces se duplica el descriptor de archivo del socket al descriptor de archivo estándar de entrada (0) y al descriptor de archivo estándar de salida (1), así cualquier llamada de lectura read() y de escritura write() se realizará en el socket que representa la conexión con el cliente.
